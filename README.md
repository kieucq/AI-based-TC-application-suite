# AI-based Tropical Cyclone Research and Applications
*Chanh Kieu, Indiana University, Bloomington*

---

This repository serves as a central hub for a collection of deep learning (DL)-based applications for tropical cyclone (TC) forecasting developed in the **Laboratory of Tropical Cyclone Dynamics**, Department of Earth and Atmospheric Sciences, Indiana University Bloomington, since 2023. 

Instead of containing a single software package, this repository provides links to a number of public and/or private repositories under this GitHub account. Together, these projects address multiple aspects of TC forecasting, ranging from TC genesis prediction and intensity estimation to climate reconstruction that are based on current DL architectures. Most systems are designed for both research and real-time operational applications.

The available repositories and their primary functionalities are summarized below.

## 1. TC Intensity Retrieval

**Repository:** `Deep-learning-intensity-downscaling`

This repository implements convolutional neural network (CNN) and Vision Transformer (ViT) architectures for retrieving three key TC intensity and structural parameters directly from coarse-resolution climate reanalysis or global forecast (GFS) data, which include i) Maximum sustained wind speed (VMAX), ii) Minimum sea-level pressure (PMIN), and iii) Radius of maximum wind (RMW). The models can be used for TC intensity downscaling or real-time intensity forecasting.

**References:** Luong and Kieu (2026); Luong (2026)

## 2. TC Genesis Forecast

**Repository:** `tcg_deep_learning`

This repository contains three deep learning architectures (CNN, ResNet, and U-Net) for predicting tropical cyclone genesis from climate reanalysis datasets. The models formulate genesis prediction as a binary classification problem (formation versus non-formation) and support both fixed-domain and basin-wide applications. The framework is suitable for real-time tropical cyclone genesis forecasting.

**References:** Nguyen and Kieu (2023); Kieu and Nguyen (2024); Nguyen (2024)

## 3. TC Regional Models

**Repository:** `regional-TCAI-models`

This repository provides a TC-following regional forecast system capable of producing three-dimensional atmospheric forecasts in a manner similar to an operational regional numerical weather prediction (NWP) model. The main architectures include physics-informed neural network (PINN), convLSTM, diffusion model, and vision transformer, which are currently used for the real-time experiment at www.tchunting.net.

The moving-domain framework continuously updates lateral boundary conditions as the computational domain follows the TC along its forecast track. In addition to real-time TC forecasting, the system can also be configured as a conventional limited-area weather prediction model over a fixed domain.

**References:** Kieu (2024); Kieu et al. (2026); Luong (2026)

## 4. TC RI Prediction

**Repository:** `RI-prediction-machine-learning`

This repository contains a machine learning framework for predicting tropical cyclone rapid intensification (RI) as a binary event. The model uses SHIPS environmental predictors generated by regional forecast systems such as HWRF or HAFS. Provided that the required environmental predictors are available, the framework can be readily applied to real-time RI forecasting.

**References:** Lowry and Kieu (2024)

## 5. TC Climate Reconstruction

**Repository:** `tcg-net`

This repository extends the tropical cyclone genesis prediction framework to basin-scale applications. Using a sliding-window approach, the model performs continuous scans over an entire ocean basin to generate spatial maps of tropical cyclone formation probability. The system can also reconstruct climatological distributions of genesis over extended periods, making it suitable for climate analysis as well as operational forecasting.

**References:** Le et al. (2025); Kieu et al. (2026)
